# Hazel Engine 
A simple C++ game engine built from scratch following The Cherno's Hazel series. 
 
## ?? Features 
- Modular C++ architecture 
- Premake build system 
- Event system (keyboard, mouse, window) 
- Logging via spdlog 
 
## ??? Build Instructions 
1. Run `GenerateProjects.bat` to generate Visual Studio solution. 
2. Open `Hazel.sln` in Visual Studio. 
3. Build and run the `Sandbox` project.
